# Screenshots

## Recommended captures

| Filename | Description |
|----------|-------------|
| `viewer.png` | Viewer tab with detection overlay on a video frame |
| `evaluation.png` | Evaluation tab showing mAP results for multiple models |
| `analysis.png` | Analysis tab with FP/FN breakdown or confidence curves |
| `dataset_explorer.png` | Dataset Explorer with thumbnail gallery and filters |
| `model_compare.png` | Side-by-side model comparison with slider |
| `embedding_viewer.png` | t-SNE or UMAP scatter plot with class coloring |
| `augmentation.png` | Augmentation preview with mosaic output |
| `quality_checker.png` | Image quality checker results table |
| `batch_inference.png` | Batch inference export in progress |

## Tips

- Use **1920×1080** or higher resolution
- Capture with a loaded model and real data for best visual impact
- Dark backgrounds render better on GitHub dark mode
