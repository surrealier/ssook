## 추론

### 실시간 Viewer

ONNX 모델을 로드하고 비디오 또는 이미지를 열어 추론 결과를 실시간으로 확인합니다.

- Detection, Classification, Segmentation 모델 지원
- YOLO v5/v8/v9/v11, CenterNet (Darknet), 커스텀 ONNX 모델
- 컨트롤: 재생/일시정지, 프레임 건너뛰기, 속도 조절, 탐색 바
- 스냅샷 캡처 및 크롭 저장
- 박스 두께, 레이블 크기, 신뢰도 임계값 설정 가능
- 고정 배치 모델 (예: batch=4) 자동 감지 및 처리

### Model A/B Compare (모델 A/B 비교)

두 모델을 동일한 이미지에 실행하고 슬라이더로 결과를 나란히 비교합니다.

### Benchmark (벤치마크)

상세한 통계와 함께 추론 성능을 측정합니다:

- **FPS**: 초당 프레임 수
- **Latency (지연시간)**: P50, P95, P99 백분위수
- **시스템 사용량**: 추론 중 CPU 및 GPU 활용률
- **내보내기**: 시스템 정보와 결과를 CSV로 내보내기

## 데이터 관리

### Dataset Explorer (데이터셋 탐색기)

여러 보기 모드로 데이터셋을 탐색하고 분석합니다:

- **파일 목록**: 바운딩 박스 수가 표시된 이미지 썸네일
- **클래스 분포 (박스 기준)**: 클래스별 박스 수
- **클래스 분포 (이미지 기준)**: 각 클래스를 포함하는 이미지 수
- **박스 크기 분포**: Small / Medium / Large 분류
- **종횡비 분포**: 너비/높이 비율 히스토그램

필터: 다중 클래스 체크박스 필터, 박스 수 필터 (>=, =, <=). 이미지를 더블클릭하면 바운딩 박스 오버레이와 함께 볼 수 있습니다.

### Dataset Splitter (데이터셋 분할기)

데이터셋을 train/val/test 세트로 분할합니다:

- **Random Split (랜덤 분할)**: 셔플 후 비율에 따라 분할
- **Stratified Split (층화 분할)**: 분할 간 클래스 비율 유지
- 커스텀 비율 (예: test 세트를 건너뛰려면 0.8/0.2/0.0)
- 진행률 추적

### Format Converter (형식 변환기)

어노테이션 형식을 일괄 변환합니다:

- **YOLO** ↔ **COCO JSON** ↔ **Pascal VOC XML**
- 클래스 매핑 자동 처리
- 재귀적 폴더 지원

### Class Remapper (클래스 리매퍼)

클래스 ID를 일괄적으로 재매핑, 병합 또는 삭제합니다:

- 클래스 ID 5 → 0으로 변경
- 클래스 3과 4를 클래스 2로 병합
- 클래스 7의 모든 어노테이션 삭제
- 재귀적 폴더 지원

### Dataset Merger (데이터셋 병합기)

여러 데이터셋을 하나로 결합합니다:

- 여러 소스 디렉토리의 이미지와 레이블을 병합
- **dHash 중복 탐지**: 지각적으로 유사한 이미지를 식별하여 중복 방지
- 유사도 임계값 설정 가능

### Label Anomaly Detector (레이블 이상 탐지기)

문제가 있는 어노테이션을 찾습니다:

- **범위 초과 박스**: 이미지 가장자리를 넘어가는 바운딩 박스
- **크기 이상치**: 비정상적으로 작거나 큰 박스
- **과도한 겹침**: 중복일 수 있는 높은 겹침의 박스

### Image Quality Checker (이미지 품질 검사기)

이미지 품질 문제를 감지합니다:

- **블러 감지**: 라플라시안 분산이 임계값 미만
- **밝기 문제**: 너무 어둡거나 너무 밝은 이미지
- **과다 노출**: 색이 바랜 이미지
- **비정상 종횡비**: 비정상적으로 넓거나 긴 이미지

### Leaky Split Detector (분할 누수 탐지기)

train/val/test 분할 간 중복 이미지를 찾습니다. 동일한 이미지가 학습 세트와 검증 세트에 모두 존재하면 평가 결과를 신뢰할 수 없습니다.

- 지각적 해싱을 사용하여 분할 디렉토리 간 이미지 비교
- 중복 쌍과 위치를 보고

### Similarity Search (유사 이미지 검색)

아무 이미지나 쿼리하여 지각적 해싱을 사용해 데이터셋에서 가장 유사한 상위 K개 이미지를 찾습니다.

### Augmentation Preview (증강 미리보기)

데이터셋에 적용하기 전에 증강 효과를 미리 봅니다:

- **Mosaic**: 4개 이미지를 하나로 결합
- **Flip**: 수평/수직 반전
- **Rotate**: 임의 각도 회전
- **Albumentations**: 다양한 변환

## 설정

설정은 `settings/app_config.yaml`에 저장되며 세션 간에 유지됩니다:

- **모델 유형**: YOLO, CenterNet, Classification, Segmentation, CLIP, Embedder, Custom
- **신뢰도 임계값**: 최소 탐지 신뢰도 (기본값: 0.25)
- **표시 옵션**: 박스 두께, 레이블 크기, 레이블 및 신뢰도 표시/숨기기
- **클래스별 스타일**: 클래스별 색상 및 표시 설정
- **언어**: English / Korean
- **테스트 모델 다운로드**: Settings 탭에서 샘플 모델과 테스트 데이터 다운로드
