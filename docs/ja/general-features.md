## 推論

### Real-time Viewer（リアルタイムビューア）

ONNXモデルを読み込み、動画や画像を開いて推論結果をリアルタイムで確認できます。

- Detection（検出）、Classification（分類）、Segmentation（セグメンテーション）モデルに対応
- YOLO v5/v8/v9/v11、CenterNet（Darknet）、カスタムONNXモデルをサポート
- コントロール：再生/一時停止、フレームスキップ、速度調整、シークバー
- スナップショットキャプチャとクロップ保存
- ボックスの太さ、ラベルサイズ、信頼度閾値の設定が可能
- 固定バッチモデル（例：batch=4）は自動的に検出・処理

### Model A/B Compare（モデルA/B比較）

2つのモデルを同じ画像で実行し、スライダーで結果をサイドバイサイドで比較できます。

### Benchmark（ベンチマーク）

詳細な統計情報で推論性能を測定：

- **FPS**：1秒あたりのフレーム数
- **Latency（レイテンシ）**：P50、P95、P99パーセンタイル
- **システム使用率**：推論中のCPUおよびGPU使用率
- **エクスポート**：システム情報と結果をCSVに出力

## データ管理

### Dataset Explorer（データセットエクスプローラ）

複数のビューモードでデータセットを閲覧・分析：

- **ファイルリスト**：バウンディングボックス数付きの画像サムネイル
- **クラス分布（ボックス別）**：各クラスのボックス数
- **クラス分布（画像別）**：各クラスを含む画像数
- **ボックスサイズ分布**：Small / Medium / Largeの内訳
- **アスペクト比分布**：幅/高さ比のヒストグラム

フィルター：マルチクラスチェックボックスフィルター、ボックス数フィルター（>=、=、<=）。画像をダブルクリックするとバウンディングボックスオーバーレイ付きで表示されます。

### Dataset Splitter（データセットスプリッター）

データセットをtrain/val/testセットに分割：

- **Random split（ランダム分割）**：シャッフルして比率で分割
- **Stratified split（層化分割）**：分割間でクラス比率を維持
- カスタム比率（例：0.8/0.2/0.0でテストセットをスキップ）
- 進捗トラッキング

### Format Converter（フォーマットコンバータ）

アノテーションフォーマットのバッチ変換：

- **YOLO** ↔ **COCO JSON** ↔ **Pascal VOC XML**
- クラスマッピングを自動処理
- 再帰的フォルダサポート

### Class Remapper（クラスリマッパー）

クラスIDの一括リマップ、マージ、削除：

- クラスID 5 → 0に変更
- クラス3と4をクラス2にマージ
- クラス7のすべてのアノテーションを削除
- 再帰的フォルダサポート

### Dataset Merger（データセットマージャー）

複数のデータセットを1つに統合：

- 複数のソースディレクトリから画像とラベルをマージ
- **dHash重複検出**：知覚的に類似した画像を識別して重複を回避
- 類似度閾値の設定が可能

### Label Anomaly Detector（ラベル異常検出器）

問題のあるアノテーションを検出：

- **範囲外ボックス**：画像の端を超えるバウンディングボックス
- **サイズ外れ値**：異常に小さいまたは大きいボックス
- **過度な重なり**：重複の可能性がある高度に重なったボックス

### Image Quality Checker（画像品質チェッカー）

画像品質の問題を検出：

- **ブラー検出**：ラプラシアン分散が閾値以下
- **明るさの問題**：暗すぎるまたは明るすぎる画像
- **露出オーバー**：白飛びした画像
- **異常なアスペクト比**：異常に横長または縦長の画像

### Leaky Split Detector（リーキースプリット検出器）

train/val/testスプリット間の重複画像を検出。同じ画像がトレーニングセットとバリデーションセットの両方に存在する場合、評価結果は信頼できません。

- 知覚的ハッシュを使用してスプリットディレクトリ間の画像を比較
- 重複ペアとその場所を報告

### Similarity Search（類似画像検索）

任意の画像をクエリとして、知覚的ハッシュを使用してデータセット内のtop-K最類似画像を検索します。

### Augmentation Preview（オーグメンテーションプレビュー）

データセットに適用する前にオーグメンテーション効果をプレビュー：

- **Mosaic（モザイク）**：4枚の画像を1枚に結合
- **Flip（反転）**：水平/垂直
- **Rotate（回転）**：任意の角度
- **Albumentations**：各種変換

## Settings（設定）

設定は `settings/app_config.yaml` に保存され、セッション間で永続化されます：

- **モデルタイプ**：YOLO、CenterNet、Classification、Segmentation、CLIP、Embedder、Custom
- **信頼度閾値**：最小検出信頼度（デフォルト：0.25）
- **表示オプション**：ボックスの太さ、ラベルサイズ、ラベルと信頼度の表示/非表示
- **クラス固有のスタイル**：クラスごとの色と表示設定
- **言語**：English / Korean
- **テストモデルダウンロード**：Settingsタブからサンプルモデルとテストデータをダウンロード
