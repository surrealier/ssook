## 推理

### Real-time Viewer（实时查看器）

加载 ONNX 模型并打开视频或图像，即可实时查看推理结果。

- 支持 Detection（检测）、Classification（分类）和 Segmentation（分割）模型
- 支持 YOLO v5/v8/v9/v11、CenterNet（Darknet）及自定义 ONNX 模型
- 控制功能：播放/暂停、跳帧、速度调节、进度条拖动
- 截图捕获和裁剪保存
- 可配置边界框粗细、标签大小、置信度阈值
- 自动检测并处理固定批次模型（如 batch=4）

### Model A/B Compare（模型 A/B 对比）

在相同图像上运行两个模型，通过滑块并排对比结果。

### Benchmark（基准测试）

通过详细统计数据测量推理性能：

- **FPS**：每秒帧数
- **Latency（延迟）**：P50、P95、P99 百分位数
- **系统占用**：推理期间的 CPU 和 GPU 利用率
- **导出**：系统信息和结果导出为 CSV

## 数据管理

### Dataset Explorer（数据集浏览器）

以多种视图模式浏览和分析数据集：

- **文件列表**：带边界框数量的图像缩略图
- **类别分布（按框）**：每个类别有多少个边界框
- **类别分布（按图像）**：每个类别出现在多少张图像中
- **边界框尺寸分布**：Small / Medium / Large 分类
- **宽高比分布**：宽高比直方图

筛选功能：多类别复选框筛选、边界框数量筛选（>=、=、<=）。双击图像可查看带边界框叠加的大图。

### Dataset Splitter（数据集拆分器）

将数据集拆分为 train/val/test 集：

- **随机拆分**：打乱后按比例划分
- **分层拆分（Stratified）**：在各拆分集中保持类别比例
- 自定义比例（如 0.8/0.2/0.0 跳过测试集）
- 进度跟踪

### Format Converter（格式转换器）

批量转换标注格式：

- **YOLO** ↔ **COCO JSON** ↔ **Pascal VOC XML**
- 自动处理类别映射
- 支持递归文件夹

### Class Remapper（类别重映射）

批量重映射、合并或删除类别 ID：

- 将类别 ID 5 → 0
- 将类别 3 和 4 合并为类别 2
- 删除类别 7 的所有标注
- 支持递归文件夹

### Dataset Merger（数据集合并器）

将多个数据集合并为一个：

- 从多个源目录合并图像和标签
- **dHash 重复检测**：识别感知上相似的图像以避免重复
- 可配置相似度阈值

### Label Anomaly Detector（标签异常检测器）

查找有问题的标注：

- **越界框**：超出图像边缘的边界框
- **尺寸异常值**：异常小或异常大的框
- **过度重叠**：高度重叠的框，可能是重复标注

### Image Quality Checker（图像质量检查器）

检测图像质量问题：

- **模糊检测**：拉普拉斯方差低于阈值
- **亮度问题**：过暗或过亮的图像
- **过曝**：发白的图像
- **异常宽高比**：异常宽或异常高的图像

### Leaky Split Detector（数据泄漏检测器）

查找 train/val/test 拆分集之间的重复图像。如果同一张图像同时出现在训练集和验证集中，评估结果将不可靠。

- 使用感知哈希跨拆分目录比较图像
- 报告重复对及其位置

### Similarity Search（相似度搜索）

查询任意图像，使用感知哈希在数据集中找到最相似的 Top-K 图像。

### Augmentation Preview（增强预览）

在应用到数据集之前预览增强效果：

- **Mosaic（马赛克）**：将 4 张图像组合为一张
- **Flip（翻转）**：水平/垂直
- **Rotate（旋转）**：任意角度
- **Albumentations**：多种变换

## Settings（设置）

配置存储在 `settings/app_config.yaml` 中，跨会话持久保存：

- **模型类型**：YOLO、CenterNet、Classification、Segmentation、CLIP、Embedder、Custom
- **置信度阈值**：最低检测置信度（默认：0.25）
- **显示选项**：边界框粗细、标签大小、显示/隐藏标签和置信度
- **类别专属样式**：逐类别颜色和显示设置
- **语言**：English / Korean
- **测试模型下载**：在 Settings 标签页下载示例模型和测试数据
